# Fullstack Project (React + Express)

Это fullstack-приложение, разработанное с использованием следующих технологий:
- **Frontend**: React с TypeScript, Redux Toolkit для управления состоянием, Material UI для интерфейса
- **Backend**: Express.js
- **HTTP-клиент**: Axios для взаимодействия между фронтендом и бэкендом

## Структура проекта

```
fullstack/
├── client/  # Фронтенд-приложение (React, Redux Toolkit, TypeScript)
│   ├── src/
│   │   ├── app/        # Инициализация приложения (store, providers и routes)
│   │   ├── entities/   # Бизнес-сущности (модели, API)
│   │   ├── features/   # Фичи (добавление новой статьи)
│   │   ├── shared/     # Общие компоненты, утилиты, хуки
│   │   ├── pages/      # Страницы приложения
│   │   ├── widgets/    # Композиция нескольких фич и сущностей
│   │   ├── index.tsx   # Точка входа в приложение
│   ├── public/         # Статические файлы
│   ├── package.json    # Зависимости фронтенда
│   ├── tsconfig.json   # Конфигурация TypeScript
│   ├── vite.config.ts  # Конфигурация Vite
│
├── server/  # Бэкенд-приложение (Express)
│   ├── src/
│   │   ├── controllers/  # Логика обработки запросов
│   │   ├── routes/       # Определение маршрутов
│   │   ├── models/       # Модели базы данных (если используется)
│   │   ├── middleware/   # Промежуточное ПО
│   │   ├── app.ts        # Основной файл сервера
│   ├── package.json      # Зависимости сервера
├── README.md             # Документация проекта

```


## Установка зависимостей
Перейдите в каждую из директорий (Client и Server) и установите необходимые зависимости:

```bash
    cd Client
    npm install
```
```bash
    cd ../Server
    npm install
```

## Запуск проекта
Сервер:
```bash
    cd Server
    npm start
```

Клиент:
```bash
    cd Client
    npm run dev
```
После этого клиентское приложение будет доступно по адресу http://localhost:3000, а серверное — по адресу http://localhost:5000 (или другому порту, указанному в настройках).

## Основные зависимости
- **React:** библиотека для создания пользовательских интерфейсов

- **Redux Toolkit:** набор инструментов для эффективной работы с Redux

- **TypeScript:** язык программирования, расширяющий возможности JavaScript

- **Material UI:** библиотека компонентов для React

- **Axios:** библиотека для выполнения HTTP-запросов

- **Express:** минималистичный веб-фреймворк для Node.js
